# URL Shortener (Go)

Сервис сокращения URL с сохранением алиасов (коротких кодов) и редиректом на исходные ссылки.
golang + sqlite

---

## Возможности

- Создание короткой ссылки для исходного URL.
- Поддержка пользовательских алиасов (если задан) и автогенерация алиаса (если не задан).
- Редирект `/{alias}` → `original_url`.
- Хранение соответствий `alias → url` в хранилище (in-memory/файл/БД — зависит от реализации).
- Валидация входных данных.
- Защита от коллизий алиасов (409 Conflict / ошибка бизнес-логики).
